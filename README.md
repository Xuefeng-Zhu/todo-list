todo-list
=========
## Overview
This application is basically a simple todo-list.

## Feature 
+ Register an account 
+ Login in to the applicatoin using existing account 
+ List a list of todos by status 
+ Create todo item 
+ Edit todo item 
+ Toggle todo item
